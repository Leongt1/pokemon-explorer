# Pokémon Explorer

A modern Pokémon Explorer built with **React**, **TypeScript**, and **React Query**. Browse Pokémon, search by name, and view detailed information with a fast, responsive user experience.

## ✨ Features

- 🔍 Search Pokémon by name
- 📄 View Pokémon details
- ⚡ Fast data fetching with React Query
- 🔄 Automatic caching and background refetching
- ⏳ Debounced search
- 📱 Responsive UI
- 🎨 Modern React component architecture

## 🛠️ Tech Stack

- React
- TypeScript
- React Router
- TanStack React Query
- Axios
- React Icons
- Zod (Validation)
- Vite

## 📂 Project Structure

```
src/

├── services/       # API configuration

```

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone <repository-url>
```

### 2. Navigate into the project

```bash
cd pokemon-explorer
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
